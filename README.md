## AI-Powered News Aggregator & Analysis Platform
Full-Stack Web Application | Tech: Node.js, Express, EJS, Tailwind CSS, OpenAI API

## Overview
A full-featured news aggregation platform that fetches real-time headlines from multiple categories and provides AI-driven summaries and analysis. The project showcases integration of public APIs, server-rendered templates, responsive design, and real-time AI functionality.

## Features
-News Aggregation: Fetches current headlines across categories using the NewsAPI, with both search and category-specific views.\
-AI Summarization & Analysis: Uses OpenAI to generate concise, factual summaries with headings, bullet points, and plain-language insights.\
-Interactive User Experience: “Dig Deeper” button allows users to request expanded AI analysis, with seamless client-server communication.\
-Responsive Design: Dynamic, mobile-first layout ensures readability and consistent UX across devices.\
-Backend Coordination: Manages article storage, API calls, and AI processing to maintain smooth data flow from source to interface.

## Tech Stack
-Backend: Node.js, Express\
-Frontend: EJS, Tailwind CSS, JavaScript\
-AI Integration: OpenAI API for summarization and bias analysis\
-Development Practices: RESTful routing, modular templating, responsive UI/UX

## What I Learned / Highlights
-Implemented AI-driven content processing in a full-stack application.\
-Built a maintainable, modular backend to handle API integration and real-time responses.\
-Focused on usability, clear navigation, and visual presentation of article insights.\
-Designed a system that transforms raw news data into actionable and understandable summaries for end users.
